## 组件

后端的接口提供：http://api.mcyou.cc:2023

网络服务：[okhttp](https://github.com/square/okhttp)；JSON解析组件：[moshi](https://github.com/square/moshi)

日历视图使用了组件：[kizitonwose/calendar](https://github.com/kizitonwose/Calendar)

## 功能

- [x] 登录
- [ ] 注册
- [x] 登出
- [x] 登录状态保持
- [x] 周视图
- [ ] 会议室占用情况视图
- [x] 个人头像框
- [x] 查看个人会议列表
- [x] 添加会议
- [ ] 删除会议
